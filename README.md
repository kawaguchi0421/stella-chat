# Stella Chat - 星空のナレッジベース

## 📝 概要
生成AI（Amazon Bedrock / Claude 3）を活用した、プロジェクト特化型のナレッジベース・チャットアプリケーションです。
単なる一問一答ではなく、会話の履歴を保持し、プロジェクトごとに最適化されたAIの回答ルールを設定できる機能を備えています。

## ✨ 主な機能
- **コンテキストを考慮したAIチャット**: 過去の会話履歴をプロンプトに含めることで、文脈に沿った自然で高度な回答を生成します。
- **プロジェクトごとのルール設定**: 参加するプロジェクトや目的に合わせて、AIの役割や回答の前提条件をカスタマイズ可能です。
- **開発者ダッシュボード**: GitHubやAWSマネジメントコンソールなど、開発に関連したサイトへのクイックアクセスリンクを備えています。
- **セキュアな認証機能**: Amazon Cognitoを利用し、安全なログインとユーザー管理を実現しています。

## 🏗 システム構成
完全なAWSサーバーレスアーキテクチャを採用しています。
生成AI（LLM）特有のレスポンスタイムを考慮し、API Gatewayの29秒タイムアウト制限を回避するための最適な通信経路を設計しています。

- **データベース**: Amazon DynamoDB (AIの回答ルール、会話履歴の保存)
- **AI / LLM**: Amazon Bedrock (Anthropic Claude 3)
- **認証**: Amazon Cognito

## 💡 アピールポイント
- **クラウドネイティブな最適化**: 実運用におけるLLMの推論時間を考慮し、タイムアウトのボトルネックを解消するアーキテクチャを採用。
- **フルサーバーレス構成**: スケーラビリティの確保とコスト最適化を両立。
- **実践的なプロンプトエンジニアリング**: 動的なシステムプロンプト（回答ルール）をDynamoDBで管理し、柔軟な回答制御を実現。
